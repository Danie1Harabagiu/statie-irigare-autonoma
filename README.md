# Stație de Irigare Autonomă 🌱

Sistem inteligent pentru monitorizarea și controlul irigării, bazat pe Arduino și o interfață web.

## Funcționalități
* Monitorizare nivel baterie (Sistem Solar 12V)
* Monitorizare nivel bazin apă
* Control automat/manual al pompelor
* Conexiune Web Serial/Bluetooth

## Structura Proiectului
Interfața este construită folosind HTML, CSS și JavaScript (Vanilla).
